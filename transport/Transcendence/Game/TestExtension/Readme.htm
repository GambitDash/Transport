Welcome to Transcendence!

Transcendence has been designed to be easily extended and modified by interested players. At this time, the documentation and infrastructure for these modifications are very limited. Nevertheless, we encourage you to look at the source game definitions and experiment at will.

The Transcendence Forums are a great place to ask questions about game modifications.

For more information on the game and to download the latest version, visit the Transcendence page online.

Intallation Instructions

Place the files of this .zip in a folder named "Extensions" under the game folder. When successfully installed, the item and ship contained described in Test.xml will appear in the game. In particular, the test ship will appear as a new starting player ship class.

Caveats, Warnings, and Disclaimers

Transcendence is still in development and you will very likely find bugs and limitations in the data formats.
